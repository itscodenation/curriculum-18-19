## Scope and Sequence

| Unit | Overview | Objectives <br> *Coders will be able to...*| 
|:--------|:---:|:---------|
| [**0: Welcome and Routines**](units/unit0) | 3 lessons| <ul> <li> Identify two things they are excited to learn in ScriptEd and be able to name their classmates and teachers.</li> <li> Navigate the daily session document. </li> <li> Navigate the popcode coding environment and create a Github account to share their work. </li> </ul>|
| [**1: HTML**](units/unit1) | 5 lessons <br>|<ul> <li> use correct syntax to create an an opening and a closing HTML tag.</li> <li> Use correct syntax to create simple HTML elements. </li> <li> Nest HTML elements including ordered and unordered lists. </li></li> <li> Create HTML elements with attributes to include links and images in their pages. </li></ul>|
| [**2: CSS Intro**](units/unit2) | 3 lessons <br> 1 project  |<ul> <li> Select an HTML element and change its style using CSS.</li> <li> Style a page using CSS properties and values. </li> <li> Describe the difference between 'head' and 'body' by linking Google Fonts to style their page. </li> <li> Create a website for a friend using HTML and CSS. </li> </ul>|
| [**3: HTML Attributes**](units/unit3) | 2 lessons <br> 1 project  |<ul> <li> Define attributes and use id attributes to style specific elements.</li> <li> Use class attributes to style more than one element. </li> <li> Use classes and ids to continue to style an invitation to a party. </li></ul> |
| [**4: Wireframing and Flexbox**](units/unit4) | 4 lessons <br> 1 project |<ul> <li> Identify the purpose of a webpage and design a wireframe.</li> <li> Use box model (padding, border, and margin) to manipulate the spacing of elements on a page. </li> <li> Use flexbox to align elements on a webpage. </li> <li> Use flexbox to create columns on a webpage. </li><li> Create a website for a mock client. </li></ul>|
| [**5: jQuery**](units/unit5) | 4 lessons <br>2 projects|<ul> <li> Use correct jQuery syntax to hide an element on the page when it is clicked.</li><li> Use jQuery actions that take one or more arguments, including .text and .css, when a button is clicked. </li> <li> Build a webpage that has 2 or more buttons and uses 2 jQuery actions to manipulate an element on the page.</li><li> Use various jQuery actions (show, append, prepend, slideUp, slideDown, fadeIn, and fadeOut) when a button is clicked. </li><li> Write a click handler that uses multiple jQuery actions on multiple elements.</li><li> Build a webpage that uses click handlers to manipulate different elements on the page. </li></ul>|
| [**Mid-Year Review Unit**](https://github.com/ScriptEdcurriculum/curriculum17-18/tree/master/foundations/units/midyear) | 2 lessons|<ul><li>Nagivate HackerRank and review key HTML, CSS, and jQuery concepts.</li> <li> Demonstrate their HTML, CSS, and JQuery knowledge through a Mid-Year HackerRank Challenge. </li> </ul> |
| [**6: JavaScript Intro**](units/unit6) | 4 lessons <br> 1 project  |<ul> <li> Define programming and compare Javascript to HTML and CSS. </li><li> Explain what a function is and make function calls using correct syntax.</li><li> Declare their own functions using correct syntax.</li><li> Compare and contrast parameters and arguments and use each appropriately.</li><li> Create a robot using JavaScript functions. </li> </ul>|
| [**7:Conditionals**](units/unit7) | 4 lessons <br> 1 project  |<ul><li> Create a new variable, store information in it, and read its value.</li><li> Identify strings and numbers as value types and differentiate between them.</li><li> Use operators to manipulate variables.</li> <li> Use input.val to retrieve a value from an input field.</li> <li> Use new JavaScript components to create a fortune teller app. </li> </ul>|
| [**8: JavaScript Functions 2**](units/unit8) | 3 lessons <br> 1 project |<ul> <li> Store a return value from a function and use it later.</li> <li> Identify the scope of a variable. </li> <li>Call a function that takes in a value from an input field and passes it as an argument.</li><li>Add additional functionality to the ScriptEd Robot by using input fields and return statements.	</li> </ul>|
| [**9: Conditionals**](units/unit9) | 3 lessons <br> 1 project  | <ul> <li> Identify booleans and create basic if/else conditional statements. </li> <li> Create conditional statements with multiple conditions using else-if. </li> <li> Understand and write compound conditionals. </li> <li> Create an interactive webpage using different conditional statements. </li> </ul>|
| [**10: Loops and Arrays**](units/unit10) | 4 lessons <br> 1 project  | <ul> <li> Create an array and determine its length.</li><li> Select an array element using the index.</li><li> Construct a for loop.</li> <li> Use a for loop over an array.</li> <li> Create a webpage that uses for loops to iterate over an array. </li> </ul>|
| [**End-of-Year Wrap Up**](https://github.com/ScriptEdcurriculum/curriculum17-18/tree/master/foundations/units/EOY%20Review%20Unit) | 3 lessons |<ul> <li> Review key concepts learned over the course of the year.</li> <li> Demonstrate their HTML, CSS, and JQuery knowledge through an End-of-Year HackerRank Challenge. </li><li> Complete an end-of-year survey to reflect on their ScriptEd experience and share feedback. </li> </ul>|
| [**Extension Unit: Animations and Collisions**](units/unit12) | 2 lessons <br> 1 project  |<ul> <li> Explain the structure of an HTML webpage.</li> <li> Create a webpage that contains a head and a body. </li> </ul>|
